## Build
多分特別なことをしなくてもビルド可能です。  
VisualStudio等のC#が扱える総合開発環境にてビルドしてあげてください。  
  
(AmongUs)というのはAmongUsのディレクトリパスの環境変数です。  
何故か環境変数が入っていない場合はディレクトリパスを環境変数で定義してください。
